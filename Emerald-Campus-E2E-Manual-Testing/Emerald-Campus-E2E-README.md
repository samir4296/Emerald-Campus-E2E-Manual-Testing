# Emerald Campus E2E Manual Testing

This repository contains end-to-end manual testing documentation for the Emerald Campus Template and the Storio Admin Panel / Management System.

## Project Scope

The project covers manual testing for:

1. Emerald Campus Template
2. Storio Admin Panel / Management System
